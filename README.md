# Brain Tumor Segmentation Using U-Net

This project focuses on brain tumor segmentation from MRI scans using Deep Learning techniques.

## Project Overview
The model uses a U-Net architecture with a MobileNetV2 backbone to detect and segment tumor regions in brain MRI images.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Nibabel

## Dataset
BraTS20 Dataset (Brain Tumor Segmentation Challenge)

## Features
- MRI image preprocessing
- Data augmentation
- Tumor mask generation
- U-Net segmentation model
- Dice coefficient evaluation
- Model training and testing

## Model Architecture
- U-Net
- MobileNetV2 Encoder
- Binary Segmentation

## Results
The model was trained on MRI brain scans and evaluated using Dice Coefficient and accuracy metrics.
